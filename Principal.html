Primera configuracion

el primer paso es crear un nuevo proyecto. se puede hacer desde la página principal de Google Cloud. posteriormete se deben definir los componentes del vps como lo son el procecador, ram, espacion de almacenamiento entre otros parametros. es importante que para que la vps funcione correctamente se habilite el tráfico HTTP y HTTPS marcando las casillas "Permitir HTTP" y "Permitir HTTPS" en la sección "Firewall".

Abrir y configurar en la terminal

En la consola de Google Cloud, ve a la página Instancias de VM despues Para conectarte a la VM de Linux que acabas de crear, haz clic en SSH en la fila de la VM. Para actualizar los paquetes disponibles y, luego, instalar el paquete apache2, usa el administrador de paquetes del sistema de ese sistema operativo. Si seguiste la guía de inicio rápido, se creará una VM de Ubuntu. Para actualizar una VM de Ubuntu, ejecuta el siguiente comando:
sudo apt update && sudo apt -y install apache2

Configurar git en el servidor

una vez instalado apache se puede crear una pagina web desde un proyecto git hub clonandolo en el servidor con los siguientes comandos: sudo apt update y posteriormente sudo apt install git luego se debe configurar el nombre de usuario y direccion de correo electronico con los siguientes comandos:
git config --global user.name "Tu Nombre"
git config --global user.email "tu@email.com"

Clonar el repositorio

En este punto es necesario haber configurado correctamente el repositorio que contiene tu pagina web se debe asegurar que contenga los archivos HTML y recursos necesarios para que tu sitio web funcione. Luego, en tu servidor, puedes clonar el repositorio en la ubicación de tu servidor web (por ejemplo, /var/www/html) utilizando el siguiente comando:
sudo git clone https://github.com/tuusuario/turepositorio.git /var/www/html en el comando anterior se debe utilizar el usuario y el repositorio de github

Configuracion de permisos

Es posible que sea necesario ajustar los permisos de los archivos y directorios clonados para que el servidor web (Apache) pueda acceder a ellos. Puedes hacerlo con el siguiente comando:
sudo chown -R www-data:www-data /var/www/html

Acceder al sitio wweb

una vez que se clono el repositorio y se concedieron los permisos necesarios se puede acceder al sitio web creado con la ip del servidor.
